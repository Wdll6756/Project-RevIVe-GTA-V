### Detailed Installation Steps

**Step 1: Prepare Your Game**
- Update GTA V to the latest version
- Create a backup of your current `mods` folder
- Remove any other map mods to avoid conflicts
- Ensure you have at least 60GB free space

**Step 2: Install Required Tools**
- Download and install **OpenIV** from the official website
- Open OpenIV and select GTA V as your game
- Click "Tools" → "ASI Manager" → Install ASI Loader
- Install **Script Hook V** by copying files to GTA V directory

**Step 3: Install Gameconfig (Required)**
- Download a gameconfig.xml file that supports map mods
- Navigate to `mods\update\update.rpf\common\data\`
- Replace the gameconfig.xml with the one included or downloaded
- Install heap adjuster and packfile limit adjuster

**Step 4: Download and Install Project RevIVe**
- Click the download button above
- Extract the downloaded archive (the file may be 15-30GB)
- Open OpenIV and enable Edit Mode
- Copy the `mods` folder from the archive to your GTA V directory
- Or run the automatic installer for easier setup

**Step 5: Configure Game Settings**
- Launch GTA V
- Go to Settings → Graphics
- Set Population Density to maximum
- Set Population Variety to maximum
- Set Distance Scaling to maximum
- Set Extended Distance Scaling to maximum
- Set Shadow Quality to High or Very High
- Set Reflection Quality to High or Ultra

**Step 6: Travel to Liberty City**
- Liberty City is accessed via the airport in Los Santos
- Look for the blip on your radar
- Fly to the marker to travel between maps
- Alternatively, use the included teleport script

## ❓ Frequently Asked Questions

**Q: Is Project RevIVe free?**  
A: Yes, Project RevIVe is completely free to download and use. The link above provides the full version.

**Q: Do I need GTA IV to play Project RevIVe?**  
A: No, Project RevIVe is completely standalone. You only need GTA V. No GTA IV files required.

**Q: Will Project RevIVe work in GTA Online?**  
A: **NEVER use Project RevIVe in GTA Online.** This is an offline-only mod. Using any map mod in Online will result in an immediate ban.

**Q: What's the difference between Project RevIVe and Liberty City V?**  
A: Project RevIVe is more complete, with working interiors, traffic paths, pedestrian routes, and 4K textures. Liberty City V is an older, less complete port.

**Q: Can I use graphics mods with Project RevIVe?**  
A: Yes, Project RevIVe is compatible with NaturalVision Evolved, QuantV, VisualV, and other graphics mods. Install the graphics mod first, then Project RevIVe.

**Q: What FPS can I expect with Project RevIVe?**  
A: Liberty City is more demanding than Los Santos due to dense geometry. On RTX 3070 at 1440p High: 50-70 FPS. On GTX 1060 at 1080p Medium: 40-50 FPS.

**Q: How do I travel between Los Santos and Liberty City?**  
A: Go to Los Santos International Airport. Look for the green blip on your radar. Fly to the marker to teleport between maps.

**Q: Are there missions in Liberty City?**  
A: The base mod includes exploration only. An optional add-on adds custom missions created by the community.

**Q: Does this include GTA IV vehicles?**  
A: The base mod includes map only. An optional vehicle pack adds all GTA IV cars, bikes, boats, and helicopters.

**Q: How much disk space do I need?**  
A: The full installation requires approximately 50GB of free space (15GB for base + 35GB for high-resolution textures).

**Q: Will this conflict with other map mods?**  
A: Project RevIVe may conflict with other map mods that add new areas. It is compatible with graphics mods and script mods.

**Q: How do I uninstall Project RevIVe?**  
A: Delete the `x64e.rpf` file from your `mods` folder, or restore your backup `mods` folder.
